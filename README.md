## Wotd

Shows Polish words of the day downloaded from www.nkjp.pl (Narodowy Korpus Języka Polskiego).

Pokazuje "Słowa dnia" z serwisu www.nkjp.pl. Program umożliwia wyszukiwanie pojawienia się danego słowa.
